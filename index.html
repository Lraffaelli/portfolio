PUBLICACIONES

¿Qué es React y cómo funciona?

Para crear un sitio o aplicación web que ofrezca funcionalidades complejas es necesario contar con tecnología que posibilite el desarrollo de software óptimo y eficaz. Desde controlar el clic en un botón, hasta animar el logo de una empresa.

¿Qué es Diseño UX/UI y qué hace un Diseñador UX/UI?

La experiencia del usuario, Diseño UX UI o UX UI Design, es en definición una práctica orientada a mejorar la experiencia que tienen las personas a la hora de utilizar el sitio de una marca u organización.

¿Cuál es la diferencia entre hosting y dominio?

Ambos son elementos esenciales de un sitio web. En síntesis, el nombre de dominio es la dirección del sitio, mientras que el hosting o alojamiento web proporciona el espacio y los recursos necesarios para ponerlo en marcha.

Páginas web estáticas vs páginas web dinámicas

Cuando se trata de desarrollar un sitio web lo primero que debes considerar es cómo lo quieres construir, como un sitio web estático o como un sitio web dinámico. Pero, ¿qué hace que un sitio web sea categorizado “estático” o “dinámico”?
